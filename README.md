## ✨ **Tentang Aplikasi**

**Reminder PR** adalah aplikasi web manajemen tugas sekolah yang dirancang modern, interaktif, dan siap pakai tanpa instalasi. Dengan tampilan **glassmorphism** yang elegan, aplikasi ini membantu siswa mengatur pekerjaan rumah (PR) secara efisien. Dilengkapi dengan **pelacakan deadline otomatis**, **prioritas tugas**, **sistem streak motivasi**, dan **penyimpanan lokal** – semua berjalan di browser tanpa perlu server.

> 🎯 **Cocok untuk** pelajar, guru, atau siapa pun yang ingin mengelola tugas akademik dengan gaya.

🧠 Cara Kerja

Struktur Data Tugas

Setiap tugas disimpan sebagai objek di localStorage:

```javascript
{
  id: "id_unik",
  name: "Latihan Matematika",
  subject: "Matematika",
  priority: "high",
  deadline: "2025-12-31T23:59",
  completed: false,
  completedAt: null,
  createdAt: "2025-01-01T12:00"
}
```

Logika Status

· Mendesak → Deadline < 24 jam & belum selesai
· Terlambat → Deadline sudah lewat & belum selesai
· Selesai → Status completed = true
· Belum Selesai → Semua kasus lainnya

Perhitungan Streak

· Bertambah 1 setiap hari Anda menyelesaikan setidaknya satu tugas tepat waktu.
· Akan reset ke 0 jika ada tugas yang statusnya Terlambat.
· Tugas yang diselesaikan setelah deadline tidak dihitung ke streak.

Lingkaran Progres

· SVG lingkaran dengan gradien – diperbarui dinamis berdasarkan rasio completed/total.

🎮 Panduan Pengguna

Aksi Cara Melakukan
➕ Tambah Tugas Isi form (Nama, Mapel, Prioritas, Deadline) → Klik Tambah Tugas
✅ Selesaikan Tugas Klik tombol hijau Selesai pada kartu tugas
↩️ Batalkan Selesai Klik tombol biru Batal pada tugas yang sudah selesai
🗑️ Hapus Tugas Klik tombol merah Hapus → Konfirmasi penghapusan
🔍 Cari Tugas Ketik di kotak pencarian (mencari nama & mapel)
🧹 Bersihkan Pencarian Klik ikon ✖️ di dalam kotak pencarian
🎚️ Filter Gunakan tab: Semua, Aktif, Selesai, Terlambat
🌙 Ganti Tema Klik ikon bulan/matahari di navbar
📅 Input Deadline Gunakan datetime-local picker (min = waktu sekarang)
